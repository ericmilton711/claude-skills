# Claude Code Skills

My custom Skills and Commands for Claude Code.

## Skills

- **midi-keyboard** - MIDI Keyboard 3D printing project with Cherry MX switches
- **3d-printing/midi-piano-project** - MIDI Piano design documentation
- **esp32-door-led-strip** - ESP32 door LED strip project with PlatformIO
- **esp32-cam-pan-tilt** - ESP32 camera with pan/tilt servo control
- **galaxy-watch-dock** - Galaxy watch charging dock project
- **printing-from-claude** - Printing utilities

## Commands

- **esp32-closet-lights** - ESP32 LED closet lighting system with PIR motion sensor

## Installation

Copy the contents to your Claude Code configuration directory:
- **Windows**: `%USERPROFILE%\.claude\`
- **macOS/Linux**: `~/.claude/`

```bash
# Example on Windows
cp -r skills/* %USERPROFILE%\.claude\skills\
cp -r commands/* %USERPROFILE%\.claude\commands\
```
